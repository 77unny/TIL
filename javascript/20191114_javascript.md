//데이터를 변형시키는 배열 메소드
pop() : 맨뒤 제거
unshift : 맨앞 제거
shift : 맨앞 추가
push() : 맨뒤 축가
splice() : a번째에서 b갯수 반환

//데이터를 유지하는 배열 메소드
concat() : 문자열, 배열 추가하여 더해준다.
join() : 배열을 문자열로 이어준다.
split() : 문자열을 배열로 만들어준다.
slice() : a번째에서 0에서 b번째의 바로앞까지 반환
indexOf() : 몇번쨰 배열인지 알려준다.
```
```

mkdir : 디렉토리 생성
rmdir : 디렉토리 제거
touch : 파일 생성
